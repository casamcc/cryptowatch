# CryptoWatch

Public static dashboard for crypto **narrative / ecosystem basket** rotation.

This repository publishes only the generated website. The private data pipeline,
API keys, operator notes, and full historical archive are not included.

## What this is

- Equal-weight thematic baskets (DeFi, AI, Memes, Perps, SOL/BASE ecosystems, etc.)
- Buy-and-hold between rebalances (no daily re-ranking)
- Breadth + birth-phase radar panel
- Descriptive market monitor, not investment advice

## Live site

GitHub Pages serves `index.html` from this repository root.

## Data sources

- CoinGecko (spot market data)
- Hyperliquid (funding / OI snapshots where shown)
- DexScreener (radar snapshots)

## Disclaimer

Read-only analytics for informational purposes only. Not investment advice.
No guarantee of accuracy or timeliness. Do your own research.
